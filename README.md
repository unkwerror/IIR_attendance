# Startup Studio NSU Attendance

Отметка посещаемости, которая не отнимает время у пары.

Проект создан для того, чтобы вход на занятие занимал секунды, а не минуты:
быстрый старт, понятный процесс и аккуратный опыт для преподавателя и студента.

---

### Идея

Сделать посещаемость простой, современной и спокойной:
- без лишних действий,
- без суеты в начале занятия,
- с фокусом на учебе, а не на формальностях.

### Для кого

Для команд, где важно:
- экономить время на каждом занятии,
- сохранять понятный и честный формат отметки,
- поддерживать удобный ритм работы для всей аудитории.
