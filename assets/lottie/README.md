# Lottie-анимация для загрузочного экрана

- `animation.json` — основная анимация (пути к картинкам уже исправлены на `images/`).
- Если анимация экспортирована с растрами (image layers), положите папку **images** из экспорта сюда: `assets/lottie/images/` (файлы 1.webp, 2.webp, …). Тогда Lottie подгрузит их автоматически.
- Можно вместо папки использовать один файл **.lottie** (DotLottie): положите его сюда как `animation.lottie` и в коде замените загрузку на этот файл.
